Instance Method

scroll(to:animated:completion:)

Scrolls a specific page of the contents so that it is visible in the receiver.

Declaration

func scroll(to page: Int, animated: Bool, completion: ((Bool) -> Void)? = nil)

Parameters

page

A index defining an content of the content view controller.

animated

true if the scrolling should be animated, false if it should be immediate.
