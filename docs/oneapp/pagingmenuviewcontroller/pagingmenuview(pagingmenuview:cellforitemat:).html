Instance Method

pagingMenuView(pagingMenuView:cellForItemAt:)

Asks your data source object for the cell that corresponds to the specified item in the paging menu view. You can use this delegate methid like UITableView or UICollectionVew.

Declaration

func pagingMenuView(pagingMenuView: PagingMenuView, cellForItemAt index: Int) -> PagingMenuViewCell

Return Value

A configured cell object. You must not return nil from this method.

Parameters

pagingMenuView

The paging menu view requesting this information.

index

The index that specifies the location of the item.
