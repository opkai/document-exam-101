PagingMenuViewDelegate Implementations

Topics

Instance Methods

func pagingMenuView(pagingMenuView: PagingMenuView, didSelectItemAt: Int)
Tells the delegate that the specified row is now selected.
func pagingMenuView(pagingMenuView: PagingMenuView, willAnimateFocusViewTo: Int, with: PagingMenuFocusViewAnimationCoordinator)
Notifies the menu view that the frame of its focus view is about to change. The menu view calls this method before adding a cell to its content. Use this method to detect cell additions, as opposed to monitoring the cell itself to see when it appears.
func pagingMenuView(pagingMenuView: PagingMenuView, willDisplay: PagingMenuViewCell, forItemAt: Int)
Tells the delegate that the specified cell is about to be displayed in the menu view.
