Instance Method

scroll(index:percent:animated:)

Scrolls a specific index of the menu so that it is visible in the receiver.

Declaration

func scroll(index: Int, percent: CGFloat = 0, animated: Bool = true)

Parameters

index

A index defining an menu of the menu view.

percent

A rate that transit from the index.

animated

true if the scrolling should be animated, false if it should be immediate.
